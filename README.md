# 選手解憂信箱｜運動心理支持系統

一個讓運動選手安心寫下心情、心理教練依紅黃綠燈優先回覆的單頁式 Web App。
概念類似「解憂雜貨店」，使用場景為選手、心理教練、基層教練與家長。

## 線上使用
GitHub Pages 部署後，手機/電腦開網址即可使用。

## 四個角色
- **選手端**：心情、7 項心理量表、壓力來源、解憂信箱三題，送出後顯示溫暖鼓勵。
- **心理教練後台**：紅黃綠燈分類、高風險警示、回覆、處理狀態、一鍵 AI 回覆初稿（5 型）。
- **教練端**：不顯示私密文字，只給燈號與今日帶隊建議。
- **家長端**：本週心理能量、建議說法／避免說法。

## 安全機制
偵測高風險語句（不想活、想傷害自己…）即標記並提醒立即關心。
⚠️ 本系統非醫療診斷工具，緊急請撥 1925／1995／110。

## 技術
- 純前端單檔 `index.html`（HTML/CSS/JS）
- 資料儲存：localStorage（本機）＋ 可選 Google Apps Script + Sheets 雲端後端
- 後端原始碼：`Code.gs`（部署說明見檔案開頭註解）

## 設定雲端（選用）
1. 在 Apps Script 貼上 `Code.gs`，執行 `setup()`（自動建立試算表）。
2. 部署為網頁應用程式（執行身分：我自己／存取：任何人），取得 `/exec` 網址。
3. 將網址填入 `index.html` 最上方的 `GAS_URL`。留空則只用本機 localStorage。
