# App Logo Assets

- `app-logo.png`: main app logo supplied for 選手心情補給站.
- `app-logo.svg`: editable fallback vector logo.
- `icon-192.png`: PWA install icon, referenced by `manifest.webmanifest`.
- `icon-512.png`: large PWA install icon, referenced by `manifest.webmanifest`.

Keep app icons in this folder so the manifest, service worker, and future app packages use one asset location.
